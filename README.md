# List Curator Agent

Turn a natural-language ask like "Y Combinator W24 batch — get founder contacts" into a clean, reviewed list of accounts or contacts. The agent proposes the seed sources, hands you a quick checkpoint to confirm what will be scraped, runs the discovery work, and lets you approve the final list before anything is saved.

## Capabilities

- Propose seed sources and a scrape plan from a plain-language intent
- Pause for a quick review of what will be scraped before any crawling starts
- Run web scraping and company / contact discovery end to end
- Pause for a final review of the assembled list before it is created
- Atomically materialize the approved list of accounts or contacts
